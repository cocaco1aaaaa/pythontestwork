# Система реферальных кодов

Этот проект представляет собой систему реферальных кодов, которая позволяет пользователям привлекать новых пользователей с помощью уникальных кодов.

## Функциональности

- **Регистрация и аутентификация пользователей**: Пользователи могут зарегистрироваться и войти в систему с помощью своих учетных данных.
  
- **Создание реферального кода**: При регистрации нового пользователя ему генерируется уникальный реферальный код.

- **Использование реферального кода**: Пользователи могут приглашать других пользователей, предоставляя им свой реферальный код. При регистрации нового пользователя с использованием реферального кода, пользователь, который предоставил код, может получить какую-то выгоду.

- **Удаление реферального кода**: Пользователи имеют возможность удалить свой реферальный код, если они больше не хотят им пользоваться.
